建议文件：

- `public/battle/settlement-adapter.js`

目标：

- battle 页在局部保存失败、版本冲突时给出更明确提示

建议方向：

- 明确区分“保存中 / 已保存 / 冲突 / 失败”
- 减少只在控制台输出的错误
